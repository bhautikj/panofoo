From a single image

If you have a single image, there are two quick methods for creating a vrEmbed. Lets just work with this image - http://vrembed.org/src/assets/ferry_building.jpg - as an example:

Passing it directly to vrEmbed.org

You can take your image URL and pass it directly to vrEmbed (the specific meanings for the parameters used can be found in the vrEmbed parameters section below):
https://vrEmbed.org/?src=//vrembed.org/src/assets/ferry_building.jpg&isStereo=false&sphereParams=360,180,0,0
Visit the link: http://vrembed.org/?src=//vrembed.org/src/assets/ferry_building.jpg&isStereo=false&sphereParams=360,180,0,0

Creating an embed code

You can also create an embed code to do the same, and use this fragment of HTML code wherever you like to embed the html (barring pages that don't support HTML snippets, such as facebook or twitter). See the result here.
<a class="vrEmbedPhoto" src="http://vrembed.org/src/assets/ferry_building.jpg" isStereo="false" sphereParams="360,180,0,0"></a><script async src="//vrEmbed.org/vrEmbed.min.js" charset="utf-8"></script>

Working with 3D images

If you have a left and right image combined into a single image, you can easily turn it into a vrEmbed:
<a class="vrEmbedPhoto" src="http://vrembed.org/src/assets/anchor_stereo.jpg" isStereo="true" sphereParams="100,66,0,0" texParams="0,0,0.5,1.0,0.5,0,1,1"></a><script async src="//vrEmbed.org/vrEmbed.min.js" charset="utf-8"></script>
See the result here - hit the 'fullscreen' button and use a 3D image mode (e.g. with a headset or with red-blue glasses) to see the effect. Go here for a detailed look at the parameters you need to set.
